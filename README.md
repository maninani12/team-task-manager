Team Task Manager

Description:
A simple full-stack Team Task Manager application built using Flask and SQLite.

Features:
- User Signup/Login
- Role-based access (Admin/Member)
- Create Projects
- Create Tasks
- Task Status Tracking
- Dashboard
- Railway Deployment

Technologies Used:
- Python Flask
- SQLite Database
- HTML
- Bootstrap
- Railway

How to Run:
1. Install requirements:
   pip install -r requirements.txt

2. Run application:
   python app.py
